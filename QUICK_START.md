# Quick Start Guide - Bukhosi Caption Generator

Get up and running with the Bukhosi Caption Generator in minutes!

---

## Option 1: HTML Version (Simplest - 5 Minutes)

Perfect for immediate testing. No installation required!

### Step 1: Get Your Gemini API Key
1. Visit: https://makersuite.google.com/app/apikey
2. Sign in with your Google account
3. Click "Create API Key"
4. Copy the key

### Step 2: Open the HTML File
```bash
# Double-click the file or open in your browser:
open bukhosi_caption_generator.html
```

### Step 3: Configure
1. Scroll to the "API Configuration" section
2. Paste your Gemini API key
3. Click "Save Configuration"

### Step 4: Generate Your First Caption
1. Drag & drop an image into the upload zone
2. Click "Generate Caption"
3. Wait ~10 seconds
4. Copy the caption to clipboard
5. Paste into Instagram!

## Option 2: Python Backend (Full Features - 15 Minutes)

Includes database, caption history, and better organization.

### Step 1: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 2: Set Up Environment Variables
See `ENV_SETUP.md` for detailed instructions.

Quick version:
1. Create a `.env` file in the project root
2. Add your Gemini API key:
   ```
   GEMINI_API_KEY=your_key_here
   ```

### Step 3: Run the Server
```bash
python caption_generator_backend.py
```

### Step 4: Open in Browser
Navigate to: http://localhost:5000
